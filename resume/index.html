HijUp.com

2012–2015

Front-end Engineer

I was the sole front-end engineer for HijUp. As we were a small team, I was also taking care of the information architecture, user experience design, and a little part of the visual design.

HijUp was doing a major redesign in 2013 where everything was rebuilt from scratch. We had a hope that HijUp will grow even bigger, so for this redesign we were trying to make the code base well structured in addition to just making it functional and pretty. This includes separating the user interface and code for buyer website and administrator dashboard, using component based approach for stylesheet, and implementing module based javascript. We believed that this would ease maintenance and adding new features in the future. After working for a few months we could say that we were quite successful. The new HijUp is not only pleasant to look at but also enjoyable to work on.

Bukalapak.com

2012–current

Front-end Engineer

Bukalapak is one of the biggest online marketplace in Indonesia. In Bukalapak everyone could sell and buy goods from all over Indonesia.

As a Front-end Engineer, I was involved in many projects related to Bukalapak main website. Some of the notable projects are

  • Developed the front-end for many of the earlier features of Bukalapak like shopping cart and transaction. These features still relied heavily on jQuery.
  • Restructured stylesheets, javascripts, and HTML templates for better maintenance and performance.
  • Developed a styleguide and component library for a second redesign.
  • Involved in transitioning the front-end code to use Vue JS.
  • Developed the product search web application that interacts with Bukalapak API instead of using the back-end.
  • Involved in moving from Rails asset pipeline to webpacker for front-end asset processing.
  • Developed internal libraries that are used by other engineers. This includes front-end api client and component library.
  • Developed the infrastructure for microfrontend in Bukalapak. This includes a framework for microfrontend application and a library for rendering web pages with microfrontend.
