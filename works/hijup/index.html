HijUp

View site

HijUp.com is the world’s first muslim fashion e-commerce. Started back in 2011, now it has grown to host more than fifty brands and serving many customers from around the world.

I was the only front-end engineer for HijUp from 2012 to 2014. As we were a small team, I was also taking care of the information architecture, user experience design, and a little part of the visual design.

HijUp was doing a major rewrite in 2013. We had a hope that HijUp will grow even bigger, so for this rewrite we were trying to make the code base well structured in addition to just piling up features. This includes separating the user interface and code for buyer website and administrator dashboard, using component based approach for stylesheet, and implementing module based javascript. We believed that this would make maintenance and adding new features easier in the future. After working for a few months we could say that we were quite successful. The website ran smoothly and the development experience was quite enjoyable.
