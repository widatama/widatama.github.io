HijUp

HijUp.com is the world’s first muslim fashion e-commerce started back in 2011. Now it has grown to host more than fifty brands and serving many customers from around the world.

I was the sole front-end engineer for HijUp. As we were a small team, I was also taking care of the information architecture, user experience design, and a little part of the visual design.

HijUp was doing a major redesign in 2013 where everything was rebuilt from scratch. We had a hope that HijUp will grow even bigger, so for this redesign we were trying to make the code base well structured in addition to just making it functional and pretty. This includes separating the user interface and code for buyer website and administrator dashboard, using component based approach for stylesheet, and implementing module based javascript. We believed that this would ease maintenance and adding new features in the future. After working for a few months we could say that we were quite successful. The new HijUp is not only pleasant to look at but also pleasant to develop.
